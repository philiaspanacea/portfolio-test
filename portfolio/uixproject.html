Spotify Redesign

Overview

Redesigning Spotify entailed creating user personas, a design brief, and a heuristics evaluation alongside wireframing. This working prototype showcases what Spotify would look like if there were more customization options for the user and if there were more features that weren't locked behind premium. This project entailed surveying Spotify users to find their pain points, which ones were already being addressed by Spotify and how, as well as ways to address the ones that were overlooked. This included quality of life wants such as a light mode, being able to contact customer service in-app, curating your home layout, and so on.
